# Roblox Studio File Manager Layout

This folder shows a simple Roblox Studio-style structure for the AFK system scripts.

```text
RobloxStudioFileManager/
├── ServerScriptService/
│   └── AFK_Timer.lua
├── StarterPlayer/
│   └── StarterPlayerScripts/
│       └── AFK_Movement_Warning.lua
└── ReplicatedStorage/
```

- Place AFK_Timer.lua in ServerScriptService
- Place AFK_Movement_Warning.lua in StarterPlayerScripts
